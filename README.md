# Robo-Fit

Stay fit during those long work days with a guided exercise routine.

### Local Development:

```
cd web
npm install
npm run dev
```

### Deployment:

You will need a vercel account and the vercel CLI:
https://vercel.com/docs/cli

To deploy a preview:

```
vercel
```

To deploy to production:

```
vercel --prod
```
